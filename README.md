# Queue Management System

A smart and efficient Queue Management System designed to optimize customer flow, reduce waiting time, and improve overall service experience. This system simulates real-world queue handling using a lightweight and scalable architecture.

---

## Overview

In many service-based environments such as banks, hospitals, and customer service centers, managing queues efficiently is a major challenge. This project provides a digital solution that allows seamless queue handling with real-time updates and structured data processing.

---

## Key Features

- Add users to the queue  
- Remove users after service completion  
- View real-time queue status  
- Fast and responsive system  
- Simple and clean user interface  
- Lightweight data handling using JSON  

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Data Storage: JSON file  

---

## System Architecture
Frontend (HTML/CSS/JS)
->
Backend (Node.js + Express)
->
JSON File (Data Storage)


---

## Working Flow

1. User interacts with the frontend interface  
2. Request is sent to backend via API  
3. Backend processes the queue logic  
4. Data is stored/updated in JSON file  
5. Updated queue is sent back to frontend
